仍是革命者-阿尔伯特·爱因斯坦

October 14, 2025

阿尔伯特·爱因斯坦是有史以来最著名的科学家，他是一位举世闻名的名人，当他出现在公众面前时，人们会像摇滚明星一样欢迎他。作为一名反战的狂热分子，爱因斯坦还就从妇女权利和种族主义到移民和核武器控制等问题发表了看法。但今天，他的形象已被阉割，变成了一个迷人的心不在焉的天才。事实上，他是社会变革的强大力量，也是政治行动主义的典范。电影制片人朱莉娅·纽曼利用大量罕见的档案片段、信件和新颖而富有启发性的采访，证明了阿尔伯特·爱因斯坦的社会和政治行动主义在今天与他那些才华横溢、开创性的理论同样重要。作者：First Run Features
